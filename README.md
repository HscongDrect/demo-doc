## 安装

``` bash
# 安装依赖
npm install

# 国内用户安装依赖
npm install --registry=http://registry.npm.taobao.org

# serve with hot reload at localhost:8080
npm run dev

# 开发环境运行demo文档
npm run demo

# 打包编译测试环境
npm run build:sit

# 打包编译生产环境
npm run build:prod

# 打包编译demo文档
npm run build:demo

# build for production and view the bundle analyzer report
npm run build --report
```
